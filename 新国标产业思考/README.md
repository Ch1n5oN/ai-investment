# 7月1日动力电池/电动车安全新国标落地，对锂电池产业链的影响推演

> 研究周期：2026-06-12 至 2026-07-01  
> 更新方式：按天收集信息，按产业链验证逻辑，最终沉淀成“信息如何影响金融定价”的作业稿。

## 一、核心结论

这次新国标不是普通政策利好，而是安全标准抬升带来的供给侧改革。

过去动力电池安全标准更接近“热失控前 5 分钟报警”，现在标准方向更接近“不起火、不爆炸，仍需报警”。同时，新标准新增或强化底部撞击、快充循环后外部短路、烟气危害、高压安全、一键断电等要求。也就是说，标准从“出事前提醒你跑”，升级为“尽量别出事”。

这可能改变产业链的四件事：

1. 技术门槛抬高：不能通过新国标的中低端产能承压。
2. 订单流向变化：头部电池厂、头部材料厂、检测认证机构和高压安全硬件供应商更容易集中受益。
3. 资金审美变化：市场不应只看“锂电池概念”，而要看谁解决新国标最痛的地方。
4. 后市场规范化：二手车残值评估、维修规范、电池回收利用等环节可能获得更多安全参考。

小美式翻译：这不是看谁名字里有“电池”，而是看谁能让电池在更苛刻环境下不烧、不炸、不失控。

## 二、题目真正训练什么

这道作业本质不是问“买哪个票”，而是训练一个判断框架：

**一个强制标准，如何改变产业链供需、技术门槛、订单流向和资金审美。**

如果只写“锂电池利好”，就是停留在概念层；如果能拆出强制认证、真实采购、技术壁垒、供应链集中、市场反馈，就进入产业推演层。

因此，本作业要持续跟踪五类信号：

1. 哪些企业率先通过 `GB 38031-2025` 或 `GB 18384-2025` 相关检测。
2. 哪些车企宣布车型适配新国标。
3. 哪些材料、设备和安全硬件被明确点名。
4. 哪些上市公司公告、互动平台、财报、研报出现订单验证。
5. 二级市场上涨时，涨的是正宗刚需，还是泛概念扩散。

## 三、标准变化：从报警逻辑到本体安全

以下为公开报道口径和已整理的研究要点，具体条款细节仍需以标准原文逐条复核为准。

### 1. 动力电池安全：`GB 38031-2025`

`GB 38031-2025`《电动汽车用动力蓄电池安全要求》将于 2026-07-01 实施。核心变化在于更强调电池本体安全和极端场景安全。

重点关注：

- 热扩散要求提高：不仅要报警，更要控制不起火、不爆炸。
- 底部撞击测试：对应车辆托底、碰撞、异物冲击等真实场景。
- 快充循环后外部短路测试：对应快充普及后的安全衰减问题。
- 烟气危害测试：不只看是否起火，还要看事故后的毒性与逃生风险。
- 绝缘、电气安全、系统级验证：电芯、电池包、整车之间的协同要求提高。

推演：标准越接近真实事故场景，越会倒逼电池厂重新评估材料、结构、热管理、测试认证能力。

### 2. 电动车高压安全：`GB 18384-2025`

`GB 18384-2025`《电动汽车安全要求》同样将于 2026-07-01 实施。它强化了高压系统安全，尤其是一键断电、物理断开、高压回路隔离等要求。

重点关注：

- 软件控制不足以构成最终安全冗余。
- 高压回路需要更可靠的物理断开能力。
- 事故救援、维修维护、碰撞后的电气安全要求提高。

推演：这条线不是单纯利好软件控制或 BMS，而是利好高压继电器、熔断器、pyrofuse、高压连接器、高压配电盒、断电装置等硬件冗余链条。

## 四、产业链方向一：电池材料升级

这是最核心的方向。

新国标要求更高热安全、抗撞击能力和快充后安全性，最终会倒逼材料体系升级。电芯厂和整车厂要通过测试，不能只靠口号，必须落实到材料、结构和工艺。

重点环节：

- 隔膜：耐热隔膜、涂覆隔膜、陶瓷涂层、复合隔膜。
- 电解液：阻燃添加剂、热稳定添加剂、高电压体系添加剂。
- 正负极材料：倍率性能、循环稳定性、热稳定性。
- 热防护材料：气凝胶、云母、阻燃材料、隔热垫。
- 电池包结构件：底部防护、抗冲击壳体、防热扩散结构。
- 快充相关材料：能承受多轮快充循环后的安全验证。

判断标准：

1. 是否真的帮助通过热扩散、底部撞击、快充后短路等测试。
2. 是否进入头部电池厂或车企供应链。
3. 是否具备量产能力和成本优势。
4. 是否因为标准提高而让中小供应商退出。

小美式翻译：谁能让电池在更苛刻环境下不烧、不炸、不失控，谁的价值提高。

## 五、产业链方向二：安全测试设备

这个方向容易被忽略，但很关键。

标准升级后，企业不是嘴上说安全就行，而是要测出来。测试项目越复杂，测试设备、实验室和检测认证服务的需求就越硬。

重点环节：

- 热扩散测试设备。
- 底部撞击测试设备。
- 快充循环测试设备。
- 外部短路测试设备。
- 烟气危害测试设备。
- 绝缘电阻、高压安全测试设备。
- 第三方检测认证服务。

判断标准：

1. 是否出现电池厂、车企、检测机构扩产或设备采购。
2. 是否具备新国标项目检测能力。
3. 是一次性设备采购，还是长期检测服务收入。
4. 是否有认证资质和客户资源。

小美式翻译：标准越严，检测越刚需。过不了测试，车都没法卖。

## 六、产业链方向三：高压断电装置

这对应电动车安全新国标。

官方信息里明确提到，电动汽车安全新国标要求高压回路实现物理断开，并把“一键断电”定义为物理断电装置，替代单纯依赖软件控制的方式。

重点环节：

- 高压继电器。
- 熔断器 / pyrofuse。
- 高压连接器。
- 高压配电盒。
- BMS / VCU 控制协同。
- 电流、电压、绝缘、故障检测传感器。
- 事故救援断电装置。

判断标准：

1. 是否已经进入主机厂供应链。
2. 是否能适配 800V 高压平台和快充场景。
3. 是否具备车规级认证和量产交付能力。
4. 是否从单车配置率提升中获得新增价值量。

小美式翻译：软件断电不够了，必须物理断。这是从系统逻辑安全升级到硬件冗余安全。

## 七、外延方向：废旧电池回收

新国标提高安全门槛后，二手车残值评估、保险定价、维修规范、电池回收利用等后市场环节可能获得更多安全参考。旧电池、低安全标准电池、退役电池是否会出现明确检测或替换需求，还需要继续找政策和订单证据。

这条线不一定最先涨，但中长期有产业逻辑：

- 新旧标准切换可能提高旧电池检测和评估需求。
- 退役电池梯次利用可能需要更严格安全评估。
- 回收拆解过程可能受益于更规范的安全要求。
- 头部电池厂、车企、回收企业可能形成闭环体系。

判断标准：

1. 是否有政策明确推动退役电池规范化。
2. 是否有车企、电池厂绑定回收渠道。
3. 是否有真实处理量、产能利用率和金属回收收益。
4. 是否只是“回收概念”，还是已经形成商业闭环。

## 八、对产业格局的影响

### 1. 利好方向

- 能通过新国标的头部电池厂。
- 具备安全材料升级能力的材料企业。
- 具备新测试项目能力的检测设备和认证服务企业。
- 进入车企供应链的高压物理断电硬件企业。
- 具备规范化处理能力的电池回收企业。

### 2. 承压方向

- 低端电池产能。
- 低价内卷、缺乏安全验证能力的中小厂。
- 没有认证资质、没有客户导入、只有概念表述的公司。
- 不能适配快充、高压、底部撞击、热扩散要求的旧产品。

### 3. 资金可能如何定价

第一阶段：概念扩散，锂电池整体、材料、设备、高压安全都可能被扫一遍。  
第二阶段：市场开始分辨正宗程度，订单、认证、供应链位置变重要。  
第三阶段：7月1日前后，资金会验证谁真正受益，蹭概念公司容易分化。  
第四阶段：若出现车企适配、检测订单、材料采购等实锤，强逻辑公司可能走出趋势。

## 九、每日跟踪方法

每天更新 `daily/YYYY-MM-DD.md`，按五个层级记录：

先标明当日文档状态：完整日报、当日工作台、待补复核。若行情和新增信息未补齐，不做资金审美结论。

1. 官方/政策：标准解读、宣贯会、地方通知、协会文件。
2. 产业链：材料、测试设备、高压断电、回收的新信息。
3. 公司验证：公告、互动平台、财报、研报、客户认证、订单。
4. 市场反馈：涨幅、资金流向、板块轮动、代表标的。
5. 今日判断：信息是否强化或削弱原逻辑。

日报最后一节不再只写“待补与明日待查”，而是写成“待查回看与下一步验证”。它的作用不是要求所有问题第二天清零，而是把未知项放进验证链条：

1. 昨日待查回看：无论是否查到，都要写今日状态、结论处理和下一次回看。
2. 今日新增待查：只记录今天新冒出的关键问题，并写清影响判断、优先级、回看日或触发条件。
3. 滚动硬证据池：标准原文、检测报告、订单定点等不一定能次日闭环的事项，进入滚动池；未闭环前不升级结论。
4. 明日执行清单：只保留明天真的要做、且能执行的 1-3 件事。

判断一条待查是否闭环，不只看有没有找到肯定答案。查不到直接证据、因此维持 E0/E1、降权或转入滚动池，也算当日处理闭环；真正未闭环的是没有状态、没有下一次回看点。

如果想知道某个问题应该在哪一天看结果，先看 `tracking.md` 的“待查回看索引”：那里会标明来源日报、当前处理、下一次看哪里/什么时候、闭环标准。日报解决当天，`tracking.md` 解决跨日。

同时增加小美体系三要素视角。新版小美 skill 里的三要素不是孤立三栏，而是一个动态系统：先用事实账本分清事实、推断、市场反馈和未知项，再看竞争格局决定方向、流动性决定行情载体、情绪位置决定交易节点。

1. 事实账本：今天新增的是事实、推断、市场反馈，还是未知项。
2. 竞争格局：强制标准是否真的改变行业门槛、供应链份额和比较优势。
3. 产业命门/正宗度：公司证据处在 E0、E1、E2、E2+ 还是 E3。
4. 流动性：资金是买泛锂电，还是买材料、检测、高压断电、回收中的正宗刚需。
5. 情绪位置：题材处在发现期、扩散期、分化期、高潮期、退潮期还是证据升级期。
6. 风险链：标准、订单、配置率、流动性和情绪中哪一环可能证伪原判断。

跟踪时最重要的是区分三类公司：

- 正宗刚需：标准变化直接带来认证、采购或配置率提升。
- 间接受益：在产业链里有位置，但订单弹性还要验证。
- 蹭概念：名字相关，但没有标准适配、客户、产能和订单证据。

证据先按四档记录：

- E0：待验证。只有题材、帖子、产业链联想，未找到公司公开材料。
- E1：公开报道线索。有媒体、地方政府转载、产业报道提到，但尚未找到公司公告或检测报告。
- E2：公司公开材料。公司公告、官网、财报、互动平台、发布会原文可验证。
- E2+：强公司公开材料 / E3 候选。公司官网或权威媒体展示检测报告/认证信息图，但完整报告 PDF、编号验真、检测机构原文仍待补。
- E3：硬验证。完整检测报告、可验真认证文件、订单、定点、客户采购等直接证据。

## 十、当前阶段判断

截至 2026-07-01，作业已经从框架建立期进入“落地日分化验证期”。

目前确定的是：新国标具有强制性，安全门槛明显抬高，材料、测试设备、高压断电三条线具备产业推演价值；废旧电池回收仍是中长期外延方向。6-23 至 7-01 这段新增数据把一个关键结论坐实了: 7-01 不是锂电大贝塔日，而是材料重点样本、检测执行入口和命门型高压硬件被市场优先定价。

已经出现的验证线索包括：宁德时代官网披露多款产品获得或通过 `GB 38031-2025` 检测报告，比亚迪官网和公开报道披露刀片电池/闪充刀片电池通过或超越新国标测试；公司观察池已把宁德时代、比亚迪作为 E2+ / E3 候选记录。后续仍要继续查检测机构官网原文、完整检测报告 PDF、报告编号验真入口和具体试验项，补齐前不写成 E3 硬验证。

目前不确定的是：哪些检测服务、测试设备、高压物理断开硬件和材料公司能真正拿到订单，哪些需求会先爆发，高压断电的单车价值量和配置节奏如何，材料/回收方向是否能从 E0 观察升级到 E1/E2/E2+ 证据，以及小美“7-01 是汽车股节点”这句话会不会被更广泛的汽车链反馈验证。

所以接下来的重点不是急着给股票答案，而是持续找证据：

- 谁通过了检测？
- 谁拿到了订单？
- 谁被车企、电池厂点名？
- 谁的产品是新国标不得不采购的东西？
- 谁只是名字沾边？

阶段性市场反馈：

- 6-16：题材发现期，多方向试探，材料、设备、高压、回收都被扫到。
- 6-17：高压连接和回收弹性最强，ETF 没有同步大涨，说明不是板块贝塔。
- 6-18：检测服务和高压物理断开硬件补涨，广电计量、苏试试验、宏发股份、中熔电气成为节后证据验证重点。
- 6-19 至 6-21：休市，重点转向证据复盘和最终稿结构整理。
- 6-22：节后锂电 ETF 与材料回潮，但检测和高压回落，更像短线贝塔修复。
- 6-23 至 6-26：回撤、修复、再杀跌反复出现，说明没有硬证据时，题材在高波动市场里不会线性兑现。
- 6-29：材料样本先行收敛，多氟多一字涨停，资金先押重点样本而不是押锂电整体。
- 6-30：氟化工作业贴把多氟多抬成重点样本，材料、检测、高压和部分回收同步预热修复。
- 7-01：标准正式实施；多氟多、苏试试验、广电计量、宏发股份走强，而电池 ETF 和宁德时代回落，说明市场优先定价的是执行入口而不是板块贝塔。

## 十一、小美式结尾

这个方向不是看谁名字里有“电池”，而是看谁解决了新国标最痛的地方。

安全标准一提高，行业就会重新分层。低端产能是压力，技术壁垒是机会。真正的机会不在热闹里，在标准提高后不得不采购、不得不认证、不得不升级的地方。
